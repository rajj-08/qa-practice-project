# QA Practice Project

Manual and exploratory testing project on 3 public web applications.

## What I tested
- Cross-browser testing (Chrome, Firefox, Edge)
- Exploratory & smoke testing
- API inspection using Chrome DevTools
- Bug documentation with severity labels & reproduction steps

## Tools Used
- Chrome DevTools (Network tab, Console)
- Browser: Chrome, Firefox, Edge
- Bug tracking: Markdown-based reports

## Results
- 8+ bugs documented with full reproduction steps
- Test cases written for core user flows
- Screenshots attached for all critical bugs
